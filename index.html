External Resources

            

                

                    CSS Reset
                

                

                    This public domain CSS reset is a great tool to make sure all your websites look
                    the same regardless of browser or device, and to make creating a stylesheet much
                    easier.
                

                
            

            

                

                    Prism
                

                

                    Prism is a lightweight, extensible syntax highlighter, built with modern web
                    standards in mind. It is a great tool for highlighting code snippets in your web
                    pages.
                

                
            

            

                

                    CSS Color Palette Generator
                

                

                    This very useful tool can help to create a good looking color palette for your
                    website. Simply choose a base color, adjust the hue shift for the secondary
                    color, saturation for neutral colors, and light/dark mode. It also allows you to
                    preview the palette so you can make sure it looks good.
                

                
            

            

                

                    Coolors
                

                

                    Coolors is a color scheme generator that helps you create beautiful color
                    palettes. Press the spacebar to refresh colors, and you can lock colors you
                    like. You can also export your palette in various formats, including CSS, PNG,
                    and PDF.
